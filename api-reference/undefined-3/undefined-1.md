<summary><mark style="color:blue;">[07.31]</mark> 업데이트 내용</summary>

#### <mark style="background-color:yellow;">상대방 프로필 조회</mark> (/user/{nickname})

* 상대방 프로필 정보를 조회한다.
  * **id** : 사용자 식별 값
  * **nickname** : 닉네임
  * **avatar** : 이미지 식별 값
  * **gameRecord** : 상대방 게임 정보
  * **gameRecord.win** : 승수
  * **gameRecord.loss** : 패수
  * **gameRecord.ladderLevel** : 래더 레벨
  * **gameRecord.achievement** : 래더 업적들
  * **isFriend** : 친구 여부
  * **isBlock** : 차단 여부

</details>
